1.Ensure the registry key is added into the system
2.Ensure the LogDeleter.exe and Logger.ini is copied to the same directory as reflected in the registry key
3.Configure the Logger.ini file to show the location of the folder to delete, and the no. of days for log files to exist
3.Run LogDeleter /install   in command
4.Go to Services -> IRUM_LogDeleter and press Play
